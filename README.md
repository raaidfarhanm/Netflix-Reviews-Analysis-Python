# Netflix Reviews Analysis Using Python

## Project Overview

This project analyzes Netflix user reviews to understand customer satisfaction, rating trends, and user engagement. The analysis was performed using Python in Jupyter Notebook with data visualization techniques.

## Objectives

* Analyze user ratings of the Netflix application.
* Identify positive and negative review patterns.
* Measure customer engagement through thumbs-up counts.
* Generate insights from large-scale review data.

## Tools and Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Dataset

The dataset contains Netflix user reviews with information such as:

* Review ID
* User Name
* Review Content
* Rating Score
* Thumbs Up Count
* App Version
* Review Date

## Analysis Performed

1. Data Cleaning
2. Missing Value Analysis
3. Rating Distribution Analysis
4. Average Rating Calculation
5. Positive vs Negative Review Analysis
6. Most Helpful Reviews Analysis
7. Review Trends Visualization

## Key Findings

* Average Netflix Rating: 2.88/5
* User ratings show mixed satisfaction levels.
* Positive and negative review patterns reveal customer experience trends.
* Highly liked reviews provide valuable user feedback for improvement.

## Conclusion

The project demonstrates the use of Python for data cleaning, exploratory data analysis (EDA), and visualization. The findings can help understand customer sentiment and improve decision-making through data-driven insights.
